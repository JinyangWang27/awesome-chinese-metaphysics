## 六爻 (Liu Yao)

六爻是以《易经》为基础的中国古典占卜方法，通过掷钱或蓍草起卦，形成由六条爻组成的卦象，进而进行解读。

- [ichingpy](https://github.com/philosopherdog/ichingpy) - 用于六爻占卜的 Python 库，提供起卦与卦象解释工具。
- [iching-js](https://github.com/placeholder/iching-js) - 用于《易经》起卦与卦象查询的 JavaScript 库。*(占位符)*
- [liuyao-core](https://github.com/placeholder/liuyao-core) - 六爻排盘与六线分析的核心引擎。*(占位符)*

## 紫微斗数 (Zi Wei Dou Shu)

紫微斗数是中国占星术的一种，通过星曜位置将出生信息映射到十二宫格中，以紫微星为核心进行命盘解读。

- [iztro](https://github.com/sylarlong/iztro) - 用于紫微斗数排盘的 JavaScript/TypeScript 库，提供宫位、星曜及大限流年数据。
- [py-iztro](https://github.com/placeholder/py-iztro) - iztro 的 Python 移植版，用于紫微斗数排盘与解读。*(占位符)*
- [zwds-calculator](https://github.com/placeholder/zwds-calculator) - 带交互式宫格展示的网页版紫微斗数排盘工具。*(占位符)*

## 八字 (BaZi)

八字（又称四柱命理）是中国命理学体系，根据出生的年、月、日、时，结合天干地支推算出四柱命盘，并进行分析解读。

- [bazi-calculator](https://github.com/placeholder/bazi-calculator) - 用于生成和分析八字四柱命盘的 Python 库。*(占位符)*
- [bazi-js](https://github.com/placeholder/bazi-js) - 用于八字排盘的 JavaScript 库，包含干支循环与十神分析。*(占位符)*
- [bazi-api](https://github.com/placeholder/bazi-api) - 提供 JSON 输出的八字排盘 REST API 服务。*(占位符)*

## 奇门遁甲 (Qi Men Dun Jia)

奇门遁甲是中国高级术数体系，用于占卜、谋略与择日，以九宫格为基础，结合天干、门、星与神灵进行布局解析。

- [qimen-sdk](https://github.com/placeholder/qimen-sdk) - 用于根据指定日期和时间构建奇门遁甲盘的 SDK。*(占位符)*
- [qmdj-engine](https://github.com/placeholder/qmdj-engine) - 用于奇门遁甲九宫布局与解读的 Python 引擎。*(占位符)*
- [qimen-web](https://github.com/placeholder/qimen-web) - 支持实时九宫计算的浏览器端奇门遁甲盘查看工具。*(占位符)*

## 风水 (Feng Shui)

风水是中国通过合理布局环境以顺应自然气场（气）的传统实践，主要分为玄空飞星派（理气）和形势派两大流派。

- [flying-stars](https://github.com/placeholder/flying-stars) - 用于玄空飞星计算与布盘的 Python 库。*(占位符)*
- [fengshui-compass](https://github.com/placeholder/fengshui-compass) - 用于风水方位分析的数字罗盘工具。*(占位符)*
- [bagua-mapper](https://github.com/placeholder/bagua-mapper) - 将八卦方位映射到平面图与空间的实用工具。*(占位符)*

## 农历/节气 (Chinese Calendar)

用于处理中国农历（阴阳合历）、二十四节气、天干及地支的工具库。

- [lunar-python](https://github.com/6tail/lunar-python) - 功能全面的 Python 库，支持中国农历、节气及干支日期转换。
- [lunar-javascript](https://github.com/6tail/lunar-javascript) - 用于中国农历/公历转换及节气计算的 JavaScript 库。
- [chinese-calendar](https://github.com/placeholder/chinese-calendar) - 支持公历与农历互转及节假日查询的 Python 包。*(占位符)*
- [jieqi](https://github.com/placeholder/jieqi) - 用于计算任意年份二十四节气精确时刻的工具库。*(占位符)*
