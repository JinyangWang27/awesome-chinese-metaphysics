# Contribution Guidelines

Thank you for considering a contribution to **Awesome Chinese Metaphysics**! This list curates high-quality tools, libraries, and resources related to Chinese metaphysics systems. Please follow these guidelines to keep the list useful and consistent.

## Quality Standards

Before submitting an entry, make sure it meets **all** of the following criteria:

- **Relevant** — The project or resource is directly related to at least one of the covered systems: 六爻 (Liu Yao), 紫微斗数 (Zi Wei Dou Shu), 八字 (BaZi), 奇门遁甲 (Qi Men Dun Jia), 风水 (Feng Shui), or 农历/节气 (Chinese Calendar).
- **Functional** — Software projects must be usable (installable or runnable). Abandoned projects with no recent maintenance and no community activity are generally not accepted.
- **Documented** — The project must have at minimum a README or equivalent documentation explaining how to use it.
- **Open or Accessible** — Prefer open-source projects. Commercial products and paid learning resources may be included if they are widely recognised and genuinely valuable to practitioners.
- **Not a duplicate** — Check that a very similar entry does not already exist in the list.

## Format Rules

Each entry must follow this exact format on a single line:

```
- [Name](url) - One-line description ending with a period.
```

- Use **Title Case** for the entry name where appropriate.
- The description must start with a capital letter and end with a full stop (`.`).
- Keep the description concise — one sentence, ideally under 100 characters.
- Include the Chinese name in the entry name or description where it adds clarity (e.g. `BaZi (八字)`).
- Do **not** add trailing whitespace.
- Entries within a section should be ordered alphabetically by name.

### Example

```
- [lunar-python](https://github.com/6tail/lunar-python) - Comprehensive Python library for Chinese lunar calendar, solar terms, and stem-branch date conversion.
```

## Adding a New Section

If you believe an important Chinese metaphysics topic is missing from the list, you may propose a new section. The topic must:

1. Be a distinct and recognised system or discipline within Chinese metaphysics.
2. Have at least **three** entries ready to populate the section.
3. Be described with an introductory sentence explaining the system.

## How to Submit a Pull Request

1. **Fork** this repository and create a new branch from `main` (e.g. `add-bazi-library`).
2. Make your changes following the format rules above.
3. **Run a spell-check** on your additions.
4. Open a pull request against the `main` branch of this repository.
5. Fill in the [pull request template](.github/PULL_REQUEST_TEMPLATE.md) completely.
6. Be prepared to respond to reviewer feedback within a reasonable time.

### Checklist Before Opening a PR

- [ ] My entry follows the `- [Name](url) - Description.` format exactly.
- [ ] The linked URL is reachable and points to the correct project or resource.
- [ ] The description is one sentence, starts with a capital letter, and ends with a period.
- [ ] I have placed the entry in the correct section and in alphabetical order.
- [ ] I have read the quality standards and my entry meets all of them.
- [ ] I have not submitted a duplicate entry.

## Reporting Issues

If you find a broken link, outdated entry, or factual error, please open an issue rather than a pull request so it can be triaged first.

## Code of Conduct

Be respectful and constructive in all interactions. Contributions that demean, harass, or exclude others will not be accepted.
